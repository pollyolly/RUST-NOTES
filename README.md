# RUST-NOTES

### Installation
#### M1
```
go to https://www.rust-lang.org/tools/install
run command: $curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
Check installation
```
$which rustc
$which cargo
```
Running in Commandline M1
```
$rustc test.rs
$./test
```
Creating library
```
$cd proj_dir
$cargo new "library_name" --lib
```
